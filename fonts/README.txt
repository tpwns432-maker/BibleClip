[BibleClip] 읽기 글꼴(reading font) 폴더
========================================

이 폴더에 .ttf / .otf / .woff2 글꼴 파일을 넣으면,
앱의 ⚙ 설정 → 읽기 → "읽기 글꼴" 드롭다운에 자동으로 나타납니다.

- 파일 이름이 글꼴 이름으로 표시됩니다 (예: MyFont.ttf → "MyFont").
- 선택하면 성경 본문 글꼴이 바뀌고, 다음 실행에도 유지됩니다.
- "기본"을 고르면 기본 Pretendard 글꼴로 돌아갑니다.

※ 배포(.exe) 버전에서는 이 폴더가 실행 파일(BibleClipWeb.exe) 옆에 있습니다.
   (⚙ 설정 → 데이터 폴더 열기 로 찾아갈 수 있습니다.)
