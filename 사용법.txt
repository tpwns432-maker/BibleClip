============================================================
 BibleClip 사용법  (v1.0.9 / 웹 UI 버전)
============================================================

BibleClip은 "복사한 성경 구절을 자동으로 알아보고, 내가 정한 형식으로
바꿔서 다시 복사해 주는" 한국어 데스크톱 앱입니다.
설교 준비, 문서 작성, 메신저 공유 등에서 구절을 빠르게 인용할 때 편리합니다.

[이번 버전(v1.0.7) 새 기능]
- 카드 안정성: 새 카드를 열어도 다른 카드 내용이 사라지지 않고, 같은 장을 띄운
  카드들이 서로 끌려다니지 않습니다.
- 본문↔원어 정렬: 글꼴 종류·크기를 바꿔도 본문과 원어 카드가 같은 절에 맞춰집니다.
- 설교 장바구니 일괄 추출: 장바구니에서 [전체 추출] 또는 체크한 [선택 추출]로
  여러 성구를 한 번에 클립보드로 내보냅니다.
- 사용자 약칭: 설정(⚙) → 참조 입력 → 약칭 관리에서 나만의 책이름 약칭을 등록합니다
  (예: 1요 → 요한1서). 숫자는 책 이름 맨 앞에만 붙일 수 있습니다.


------------------------------------------------------------
 1. 설치 / 실행
------------------------------------------------------------
- 설치 과정이 따로 없습니다. 받은 압축(zip)을 풀고 실행 파일만 더블클릭하세요.
    · Windows : 폴더 안의  BibleClipWeb.exe  실행
    · macOS   : BibleClipWeb.app 실행 (또는 dmg를 열어 응용 프로그램으로 드래그)
- 별도 런타임 설치가 필요 없습니다.
    · Windows 11에는 화면 엔진(WebView2)이 기본 내장되어 있습니다.
    · macOS는 내장 웹엔진을 사용합니다.
- 폴더 안의 bible_versions, original_lang 폴더는 성경/원어 데이터입니다.
  실행 파일과 같은 위치에 그대로 두세요. (지우면 본문이 안 나옵니다.)
- 다른 성경 번역본(역본)을 추가하고 싶으면 7번 항목의
  "새 성경 번역본 추가하기"를 보세요.


------------------------------------------------------------
 2. 핵심 기능 — 클립보드 모니터링 (가장 중요)
------------------------------------------------------------
이 앱의 핵심입니다. 다른 프로그램에서 성경 구절을 "복사"만 하면 자동으로
변환된 본문으로 클립보드를 바꿔 줍니다.

사용 방법
  1) 상단의 [모니터링 시작] 버튼을 누릅니다.  (배지가 초록색 "모니터링 중"으로 바뀜)
  2) 아무 곳(메모장, 카톡, 한글, 워드 등)에서 구절 표기를 복사합니다.
       예)  요 3:16
  3) 그 즉시 클립보드 내용이 "설정한 형식의 본문"으로 바뀝니다.
     이제 원하는 곳에 붙여넣기(Ctrl+V / Cmd+V) 하면 본문이 들어갑니다.
  4) 변환되면 화면 하단에 잠깐 알림이 뜨고, 해당 구절이 카드에 표시됩니다.
  5) 다 쓰면 [모니터링 중지]를 누르면 됩니다.

인식하는 표기 예시
  창 2:1            한 절
  요 1:1-3          여러 절(범위)
  시 23:1~6         물결표(~)도 가능
  요 1:1-2,4-6      콤마로 끊어 적은 불연속 구절(1·2·4·5·6절을 한 번에 인식)
  (한글 약칭/정식, 영문 표기(Gen 1:1 · 1Ths 1:1 등) 모두 인식.
   넣어둔 역본의 책이름도 자동으로 읽어 매핑합니다.)

좌측 레일(세로 아이콘 줄)의 "클립보드" 아이콘을 누르면 그동안 인식한 구절들이
목록(활동 로그)으로 쌓입니다. 목록의 항목을 클릭하면 그 구절로 다시 이동하고
클립보드에도 다시 복사됩니다.


------------------------------------------------------------
 3. 키워드 검색
------------------------------------------------------------
- 좌측 레일의 "돋보기(검색)" 아이콘을 눌러 검색 화면으로 갑니다.
- 검색어를 입력하고 Enter 또는 [검색]을 누르면 결과가 나옵니다.
- 입력창 옆의 역본 버튼으로 "검색할 성경 역본"을 고를 수 있습니다.
  (기본값은 지금 보고 있는 역본. 바꾸면 즉시 다시 검색합니다.)
- 결과의 구절을 클릭하면 그 구절이 클립보드에 복사됩니다.
  (설정 ⚙에서 "검색 결과 클릭 시 본문으로 이동"을 켜면, 복사와 함께
   해당 구절의 본문 화면으로 함께 이동합니다.)
- 모니터링 중이라면, 다른 앱에서 "#사랑" 처럼 #을 붙여 복사하면
  자동으로 검색 화면이 열리며 결과를 보여 줍니다.


------------------------------------------------------------
 4. 성경 보기 화면 — 카드 워크스페이스
------------------------------------------------------------
성경 보기 화면은 [본문] · [원어] · [사전] 카드를 원하는 위치에 자유롭게 놓고
크기를 조절하는 "카드 작업공간"입니다. (처음 실행하면 본문 | 원어 | 사전 카드가
가로로 나란히 놓여 있습니다.)

카드 추가 / 닫기
  - 우측 상단의  ＋ 본문 · ＋ 원어 · ＋ 사전  버튼으로 카드를 새로 만듭니다.
    (화면 중앙에 살짝 겹쳐서 생깁니다. 본문 카드는 최대 4개.)
  - 각 카드 머리말 오른쪽 끝의  ✕  로 그 카드를 닫습니다.

카드 이동 / 크기 조절
  - 카드 머리말(제목 줄)을 마우스로 끌면 카드가 이동합니다.
  - 카드의 가장자리·모서리(8방향)를 끌면 크기가 조절됩니다.
  - 다른 카드나 화면 가장자리, 화면의 2·3·4등분 지점 근처에 가면
    살짝 떨어진 간격을 두고 "착" 달라붙고, 정렬 안내선이 잠깐 표시됩니다.
  - 붙어 있는 카드 쪽으로 키우면 옆 카드가 크기를 유지한 채 밀려납니다.
  - 카드는 자유롭게 겹칠 수 있고, 클릭하거나 끈 카드가 맨 앞으로 나옵니다.

활성(선택된) 카드
  - 마지막으로 클릭/조작한 카드의 테두리에 보라색 강조(네온)가 들어옵니다.
    지금 어떤 카드를 다루고 있는지 한눈에 알 수 있습니다.

책 / 장 이동
  - 카드 머리말의 책 이름·장 버튼을 눌러 선택하거나,
  - 머리말의 ‹ › 버튼으로 이전/다음 장으로 넘깁니다.
  - 키보드 ← / → 키는 "활성(보라색 강조) 카드"의 장을 넘깁니다.

카드별 뒤로/앞으로 가기  (◀ ▶)
  - 각 성경 카드 머리말 왼쪽의  ◀ ▶  로 그 카드에서 "방금 전에 보던 구절"로
    돌아가거나 다시 앞으로 갈 수 있습니다.
  - 카드마다 따로 기억하므로, 한 카드에서 뒤로 가도 다른 카드는 그대로입니다.
  - 뒤로/앞으로 가면 그때 보고 있던 절이 화면 맨 위로 다시 맞춰집니다.
  - 더 돌아갈 곳이 없으면 화살표가 흐려집니다.

여러 역본 같이 보기 (다역본 병렬)
  - 상단 역본 칩 옆 [＋] 버튼을 눌러 보고 싶은 역본을 여러 개 선택하면,
    모든 본문 카드에서 각 절이 역본별로 나란히 표시됩니다.
  - 역본 칩을 마우스로 드래그하면 표시 순서를 바꿀 수 있습니다.
  - 칩의 ✕를 누르면 그 역본을 뺍니다. (맨 앞 칩이 기준 역본)

조건부 수신 (카드 잠금)
  - 본문 카드 머리말의 자물쇠 아이콘을 누르면 그 카드를 "잠글" 수 있습니다.
    (성경 카드가 2개 이상일 때만 가능)
  - 잠긴 카드는 머리말 색조가 차분하게 바뀌고 자물쇠가 보라색으로 켜집니다.
  - 클립보드로 받은 구절은 "잠그지 않은" 첫 카드가 받아 그 위치로 이동합니다.
    단, 잠긴 카드가 마침 그 책·장을 보고 있으면 그 카드가 우선 반응합니다.
    (= 카드별로 특정 본문을 고정해 두고 보는 용도)

글자 크기 / 테마
  - 우측 상단의 A− / A+ 로 본문 글자 크기를 조절합니다.
  - 좌측 레일 맨 아래 "달" 아이콘으로 밝은/어두운 테마를 전환합니다.
    (어두운 테마일 때 달 아이콘이 노란색으로 채워집니다.)

본문에서 직접 복사
  - 본문의 절을 클릭하면 그 절이 (지금 보고 있는 역본 형식으로) 복사됩니다.
  - 여러 절을 마우스로 드래그해 선택한 뒤 떼면 그 범위가 복사됩니다.

본문 카드를 스크롤하면, 연결된 원어 카드가 같은 절로 실시간으로 따라갑니다.

※ 카드 배치(위치·크기)는 자동으로 저장되어 다음 실행 때 그대로 복원됩니다.


------------------------------------------------------------
 5. 원어(히브리어/헬라어)와 사전
------------------------------------------------------------
- [원어] 카드에는 단어마다 스트롱 번호(H/G로 시작)가 붙어 있습니다.
- 원어 카드는 머리말의 "연결" 버튼으로 어느 본문 카드를 따라갈지 정합니다.
  (연결된 본문 카드의 책·장을 그대로 보여 줍니다.)
- 스트롱 번호를 "클릭"하면, 그 카드와 연결된 [사전] 카드에 뜻풀이가 나옵니다.
    (히브리어/헬라어 원어가 큰 글씨로, 음역과 함께 표시됩니다.)
  연결된 사전 카드가 없으면 새로 하나 만들어 띄웁니다.
- 원어 단어에 마우스를 "올리면" 간단한 미리보기 풍선이 뜹니다.
- 원어 단어를 "우클릭"하면 사전 내용이 별도 창으로 열립니다.
- 상단의 한글 / 영어 토글로 사전 언어를 바꿀 수 있습니다.
  (영어 사전 데이터가 있을 때)


------------------------------------------------------------
 6. 출력 설정 (복사되는 형식 바꾸기)
------------------------------------------------------------
좌측 레일의 "슬라이더" 아이콘을 누르면 출력 설정 화면이 열립니다.

표기 설정 (한국어 역본용)
  - 책 이름      : 정식 / 약칭  (각 역본이 자기 책이름으로 표기:
                    한국어 역본 "룻기", 영어 역본 "Ruth")
  - 장절 표기    : 1:1  /  1장 1절
  - 괄호         : 없음 / [ ] / ( )
  - 표기 위치    : 본문 앞 / 본문 뒤
  - 범위 기호    : -  /  ~
  - 구분 기호    : 하이픈 / 콜론 / 띄어쓰기
  - 출력 방식    : 여러 절을 한 줄로 / 각 절을 줄마다
  - 줄마다 장:절 표시 / 버전 헤더 출력 / 장절 표기 숨기기 (켜고 끄기)

  · "한 줄로" 출력일 때, 콤마로 끊어 적은 구절(예: 요 1:1-2,4)은 묶음 사이에
    ` // ` 구분 기호가 들어갑니다.  (예) [1절] [2절] // [4절]
    연속된 절 사이에는 그대로 띄어쓰기만 들어갑니다.

출력 순서
  - 복사할 때 사용할 역본 목록입니다. ↑ ↓ 로 순서를, ✕ 로 제외,
    [＋ 역본 추가]로 추가합니다. (여러 역본이면 순서대로 함께 복사됩니다.)
  - [DB 새로고침] : bible_versions 폴더에 새 성경 파일을 넣었을 때,
    앱을 끄지 않고 다시 읽어들입니다.

새 성경 번역본(모듈) 추가하기
  - 아래 사이트에서 원하는 성경 모듈(MyBible 형식)을 내려받을 수 있습니다.
      https://www.ph4.org/b4_index.php
  - 받은 파일이 압축(zip)이면 풀고, 안의 성경 파일(.SQLite3)을
    bible_versions 폴더에 넣은 뒤 [DB 새로고침]을 누르면 바로 사용할 수 있습니다.
    (앱 설정 ⚙ 의 [데이터 폴더 열기]를 누르면 폴더가 바로 열립니다.)

미리보기
  - 설정을 바꾸면 "요 1:1-3" 예시가 실제 복사될 모습으로 즉시 갱신됩니다.

※ 여러 역본을 함께 복사하면 각 역본이 자기 언어의 책이름을 따릅니다
  (영어 성경은 영어식 표기 + 하이픈).


------------------------------------------------------------
 7. 앱 설정 (⚙ 톱니 아이콘)
------------------------------------------------------------
좌측 레일 맨 아래의 "톱니(⚙)" 아이콘을 누르면 앱 전반 설정 창이 열립니다.
(앞의 "출력 설정"은 복사 형식 설정이고, 이것은 앱 동작 설정으로 서로 다릅니다.)

  - 클립보드 확인 간격 : 빠름(0.25초) / 기본(0.5초) / 느림(1초)
      · 간격이 짧을수록 복사를 빨리 알아채지만 CPU를 조금 더 씁니다.
      · 모니터링 중에 바꿔도 즉시 반영됩니다.
  - 사전 기본 언어 : 한글 / 영어 (앱을 켤 때의 기본 사전 언어)
  - 검색 결과 클릭 시 본문으로 이동 : 켜기/끄기 (3번 참고)
  - 시작 시 자동 업데이트 확인 : 켜기/끄기
  - 표시 언어 : 한국어 / English (화면 표시 언어, 재시작 없이 즉시 전환)
  - 읽기 글꼴 : [데이터 폴더 열기] 안의 fonts 폴더에 .ttf/.otf 글꼴을 넣으면
      목록에 나타나 성경 본문 글꼴을 바꿀 수 있습니다. (기본 = Pretendard)
  - 버전 정보 / [데이터 폴더 열기] / [GitHub 열기]
  - [설정 초기화] : 모든 설정을 기본값으로 되돌립니다.
      (실수 방지를 위해 한 번 더 눌러야 실행됩니다. 누르면 앱이 새로 고쳐집니다.)


------------------------------------------------------------
 8. 업데이트
------------------------------------------------------------
- 상단 [업데이트 확인]을 누르면 새 버전이 있는지 확인합니다.
- 새 버전이 있으면 상단에 안내 띠가 뜨고,
  [지금 업데이트]를 누르면 받아서 자동으로 교체·재시작합니다.
  (또는 [릴리스 페이지]에서 직접 받을 수도 있습니다.)
- 앱을 켤 때 자동으로 한 번 조용히 확인합니다.


------------------------------------------------------------
 9. 설정 저장 / 참고
------------------------------------------------------------
- 글자 크기, 테마, 역본 선택·순서, 출력 형식, 카드 배치, 마지막으로 본 위치,
  창 크기/위치 등은 자동으로 저장되어 다음 실행 때 그대로 복원됩니다.
- 설정은 실행 파일과 같은 폴더의  bibleclip_settings.json  에 저장됩니다.
  (다른 PC로 옮길 때 이 파일을 함께 복사하면 설정이 따라옵니다.)


------------------------------------------------------------
 10. 설교 장바구니 (큐시트)
------------------------------------------------------------
설교에 쓸 성구를 미리 모아두고, 클릭 한 번으로 다시 복사하거나
본문으로 이동할 수 있습니다.

- 열기 : 좌측 레일의 "장바구니" 아이콘 → [오늘의 설교 성구] 패널.
- 담기 : 검색 결과·활동 로그 행에 마우스를 올리면 나오는 ＋, 또는
         본문 구절을 우클릭 → "설교 장바구니에 담기".
- 사용 : 담긴 항목을 클릭하면 클립보드로 다시 복사됩니다.
         (앱 설정의 "검색 결과 클릭 시 본문으로 이동"을 켜두면 본문도 함께 이동)
- 정리 : 항목에 마우스를 올려 ✕로 개별 삭제, 상단 [비우기]로 전체 삭제.
- 담긴 목록은 앱을 닫아도 유지됩니다.


------------------------------------------------------------
 11. 전체화면 프레젠테이션 (예배 · 방송 송출)
------------------------------------------------------------
- F11 : 선택한(활성) 성경 카드를 전체화면으로 띄웁니다. 잡다한 UI를 숨기고
        성구만 큰 화면에 채웁니다. (ESC 또는 다시 F11로 나가기)
- F2  : 전체화면 중 퀵 검색 창. 구절(예: 창 1:1)이나 키워드를 입력하면
        화면 속 성구가 즉시 그 구절로 전환됩니다.
- + / - : 전체화면에서 글자 크기를 키우고 줄입니다.


------------------------------------------------------------
 12. 잘 안 될 때
------------------------------------------------------------
- "모니터링 시작"이 안 되거나 복사가 안 될 때
    → 실행 파일과 같은 폴더에 bible_versions / original_lang 폴더가 있는지 확인.
    → 압축을 "폴더째" 제대로 풀었는지 확인(일부 파일만 실행하면 안 됩니다).
- 본문이 비어 보일 때
    → bible_versions 폴더에 성경 데이터(.SQLite3)가 있는지 확인.
- 창이 안 보일 때
    → 모니터가 2대 이상이면 보조 모니터에 떠 있을 수 있습니다(이전 위치를 기억함).
      다른 모니터를 확인해 보세요.
- 첫 실행이 조금 느릴 수 있습니다. 잠시 기다려 주세요.
