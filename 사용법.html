복사만 하면 성경 구절이
자동으로 변환됩니다.

BibleClip은 클립보드에 복사한 성경 구절을 자동으로 알아보고, 내가 정한 형식으로 바꿔서 다시 복사해 주는 한국어 데스크톱 앱입니다. 아래 데모는 실제 앱 화면을 그대로 재현한 것입니다.

클립보드 자동 변환구절을 복사하면 즉시 설정한 형식으로 재복사
카드 워크스페이스본문·원어·사전 카드를 자유 배치·크기 조절
다역본 병렬 보기여러 역본을 절마다 나란히, 순서는 드래그로
원어 & 사전스트롱 번호 클릭으로 히브리어·헬라어 뜻풀이
키워드 검색#키워드 복사 또는 검색 화면으로 찾기
자동 업데이트새 버전을 받아 자동으로 교체·재시작
1

설치 & 실행

받은 압축을 풀고 더블클릭하면 끝
  • 설치 과정이 따로 없습니다. 받은 압축(zip)을 풀고 실행 파일만 더블클릭하세요.
  • Windows — 폴더 안의 BibleClipWeb.exe 실행
  • macOS — BibleClipWeb.app 실행 (또는 dmg를 열어 응용 프로그램으로 드래그)
  • 별도 런타임이 필요 없습니다. Windows 11은 WebView2 기본 내장, macOS는 내장 웹엔진 사용.
데이터 폴더는 그대로 두세요. 폴더 안의 bible_versions · original_lang 는 성경/원어 데이터입니다. 실행 파일과 같은 위치에 두어야 본문이 나옵니다. 압축은 반드시 “폴더째” 풀어 주세요.
2

클립보드 모니터링

이 앱의 핵심 기능

다른 프로그램에서 성경 구절을 “복사”만 하면, 자동으로 변환된 본문으로 클립보드를 바꿔 줍니다. 아래는 실제로 일어나는 일입니다.

BibleClipv1.0.2모니터링 중
메모장에서 복사
요 3:16
클립보드가 이렇게 바뀜
KRV요 3:16 하나님이 세상을 이처럼 사랑하사 독생자를 주셨으니 이는 저를 믿는 자마다 멸망치 않고 영생을 얻게 하려 하심이니라
구절을 복사했습니다 · 요 3:16
상단 [모니터링 시작] → 배지가 초록 “모니터링 중”으로. 이후 어디서든 구절을 복사하면 즉시 변환됩니다.

인식하는 표기

창 2:1한 절
요 1:1-3여러 절(범위)
시 23:1~6물결표(~)도 가능
요 1:1-2,4-6콤마로 끊은 불연속 구절 — 1·2·4·5·6절을 한 번에 (v1.0.2 신기능)
활동 로그 — 좌측 레일의 “클립보드” 아이콘을 누르면 인식한 구절이 목록으로 쌓입니다. 항목을 클릭하면 그 구절로 다시 이동하고 클립보드에도 다시 복사됩니다.

키워드 검색

단어가 들어간 구절 찾기

좌측 레일의 돋보기 아이콘으로 검색 화면을 엽니다. 결과 구절을 클릭하면 클립보드에 복사됩니다.

검색
KRV
요일 4:8사랑하지 아니하는 자는 하나님을 알지 못하나니 이는 하나님은 사랑이심이라
요 3:16하나님이 세상을 이처럼 사랑하사 독생자를 주셨으니 이는 저를 믿는 자마다…
고전 13:4사랑은 오래 참고 사랑은 온유하며 시기하지 아니하며 사랑은 자랑하지 아니하며…
입력창 옆 역본 버튼으로 검색 역본 변경 · 결과 클릭 시 복사 · 모니터링 중엔 #사랑 복사로도 자동 검색
설정 ⚙에서 “검색 결과 클릭 시 본문으로 이동”을 켜면, 복사와 함께 그 구절의 본문 화면으로도 이동합니다.

성경 보기 — 카드 워크스페이스

본문 · 원어 · 사전 카드를 자유 배치

성경 보기 화면은 카드를 원하는 위치에 놓고 크기를 조절하는 작업공간입니다. 아래 본문 카드의 각 부분을 살펴보세요 — 보라색 테두리는 지금 선택된 활성 카드라는 뜻입니다.

2땅이 혼돈하고 공허하며 흑암이 깊음 위에 있고 하나님의 신은 수면에 운행하시니라
3하나님이 가라사대 빛이 있으라 하시매 빛이 있었고

여러 카드 자유 배치 · 활성 · 잠금

활성 카드 — 보라 네온 잠금 — 자물쇠 점등
⠿본문◀▶창세기1장✕
⠿원어✕
⠿사전연결: 본문✕
활성(선택) 카드 — 클릭한 카드 잠긴 카드 — 자물쇠 점등 + 헤더 톤 ＋ 본문/원어/사전 으로 카드 추가
  • 다른 카드·화면 가장자리·2·3·4등분 지점 근처에서 “착” 달라붙고 안내선이 표시됩니다. 붙은 카드 쪽으로 키우면 옆 카드가 밀려납니다.
  • 카드는 자유롭게 겹칠 수 있고, 클릭/드래그한 카드가 맨 앞 + 보라 네온으로.
  • 키보드 ← → 는 활성(네온) 카드의 장을 넘깁니다. (v1.0.2)

다역본 병렬 보기

상단 역본 칩으로 역본을 여러 개 켜면, 모든 본문 카드에서 절마다 역본이 나란히 표시됩니다. 칩을 드래그하면 순서가 바뀌고 맨 앞 칩이 기준 역본입니다.

KRV✕ESV✕＋ 역본 요한복음 1장
1KRV태초에 말씀이 계시니라 이 말씀이 하나님과 함께 계셨으니 이 말씀은 곧 하나님이시니라ESVIn the beginning was the Word, and the Word was with God, and the Word was God.
2KRV그가 태초에 하나님과 함께 계셨고ESVHe was in the beginning with God.
절마다 역본별 줄(KRV/ESV 배지)로 병렬 표시 · 글자 크기 A− A+

원어(히브리어/헬라어)와 사전

스트롱 번호 클릭 → 뜻풀이

원어 카드의 단어마다 붙은 스트롱 번호(H/G로 시작)를 클릭하면, 연결된 사전 카드에 뜻풀이가 나타납니다.

원어 · 창세기 1장
1태초에 H7225 하나님이 H430 천지 H8064 H776 창조하시니라 H1254
2땅이 H776 혼돈하고 H8414 공허하며 H922 흑암이 H2822
사전 · 연결: 원어
H7225 · 히브리어
רֵאשִׁית레쉬트 · rêʼshîyth
시작, 처음, 으뜸, 첫 열매. 창세기 1:1 “태초에”에 쓰인 단어로, 시간·순서의 ‘맨 처음’이나 가장 중요한 것을 뜻합니다.
H7225 처럼 스트롱 번호를 클릭 → 오른쪽 사전 카드 갱신 · 마우스 올리면 미리보기 · 우클릭은 별도 창
  • 원어 카드 머리말의 “연결” 버튼으로 어느 본문 카드를 따라갈지 정합니다.
  • 연결된 사전 카드가 없으면 새로 띄웁니다. 상단 한글 / 영어 토글로 사전 언어 전환(영어 데이터가 있을 때).

출력 설정

복사되는 형식 바꾸기

좌측 레일의 슬라이더 아이콘으로 출력 설정을 엽니다. 설정을 바꾸면 미리보기가 즉시 갱신됩니다.

출력 설정
장절 표기1:11장 1절 괄호[ ]( )없음
출력 방식한 줄로줄마다 범위 기호-~
미리보기 · 요 1:1-2,4
KRV[요 1:1-2,4] 태초에 말씀이 계시니라 이 말씀이 하나님과 함께 계셨으니 이 말씀은 곧 하나님이시니라 그가 태초에 하나님과 함께 계셨고 // 그 안에 생명이 있었으니 이 생명은 사람들의 빛이라
// 표시 (v1.0.2) — “한 줄로” 출력일 때 콤마로 끊긴 구절(1-2 / 4) 사이에 자동 삽입. 연속된 절은 띄어쓰기만.

표기 옵션 한눈에

  • 책 이름 한글/영문 · 정식/약칭  ·  표기 위치 본문 앞/뒤  ·  구분 기호 하이픈/콜론/띄어쓰기
  • 토글 — 줄마다 장:절 표시 · 버전 헤더 출력 · 장절 표기 숨기기

출력 순서 & 새 역본 추가

  • 복사에 쓸 역본 목록을 ↑ ↓ 로 정렬, ✕ 제외, ＋ 역본 추가. (여러 역본은 순서대로 함께 복사)
  • 새 번역본(MyBible 모듈)은 ph4.org 에서 받아 .SQLite3 를 bible_versions 폴더에 넣고 DB 새로고침.

※ 영어 성경(ESV/NKJV 등)은 항상 영어식 표기 + 하이픈으로 출력됩니다.

잘 안 될 때

자주 묻는 해결법
모니터링/복사가 안 될 때같은 폴더에 bible_versions · original_lang 가 있는지, 압축을 “폴더째” 풀었는지 확인.
본문이 비어 보일 때bible_versions 폴더에 성경 데이터(.SQLite3)가 있는지 확인.
창이 안 보일 때모니터가 2대 이상이면 보조 모니터에 떠 있을 수 있습니다(이전 위치 기억). 다른 모니터 확인.
첫 실행이 느릴 때처음 한 번은 조금 느릴 수 있습니다. 잠시 기다려 주세요.
