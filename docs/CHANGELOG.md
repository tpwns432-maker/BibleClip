# 변경 이력 (CHANGELOG)

BibleClip의 버전별 변경 내용입니다. 최신 버전이 위에 옵니다.

---

## v1.0.0 — BibleClip 정식 출시 🎉
- **프로젝트 이전 & 정식 1.0.0** — 기존 `AutoBible` 저장소에서 **`tpwns432-maker/BibleClip`** 으로 이전하고, 정식 버전 **1.0.0** 으로 새 출발합니다. 앱 기능은 직전 개발본과 동일하며, 저장소·자동 업데이트·릴리스 주소가 모두 새 `BibleClip` 저장소로 바뀌었습니다.
- **원격 킬 스위치(remote kill switch)** — 관리자가 배포된 앱을 원격으로 정지할 수 있습니다. 앱은 시작 시 공개 매니페스트(`killswitch.json`)를 확인하고, `disabled`가 `true`이거나 `min_version`보다 낮은 버전이면 안내 메시지만 띄우고 실행을 차단합니다.
  - **되돌리기 가능** — 데이터는 건드리지 않으며, 매니페스트를 되돌리면 다음 실행 때 정상 복구됩니다.
  - **fail-open** — 인터넷이 끊겼거나 확인에 실패하면 평소대로 실행됩니다(네트워크 장애로 전체가 멈추는 사고 방지).
  - 이 확인 코드가 포함된 **v1.0.0부터** 적용됩니다(이전 AutoBible 시절 버전은 영향 없음).
- **문서** — 사용법에 성경 번역본(MyBible 모듈) 다운로드 안내(ph4.org) 추가.

## v1.5.4
- **앱 엔진을 웹(pywebview) UI로 전환** — 리디자인 리포트의 High 단계. 기존
  CustomTkinter UI를 대체합니다. 기능은 동등하며 더 현대적인 화면으로 재구성:
  - 좌측 레일(성경 보기 · 출력 설정 · 검색 · 클립보드 로그) + 상단바 + 카드형 3분할(본문/원어/사전)
  - **클립보드 모니터링** — 복사한 구절을 자동 인식·변환·재복사, 캡처 시 본문 자동 이동·하이라이트 + 활동 로그 드로어
  - **다역본 병렬 보기** — 역본 칩 다중 선택 + 드래그로 순서 조정(라이브 애니메이션)
  - **출력 설정** — 표기/괄호/구분기호/다절 방식 등 + 출력 순서(↑↓, FLIP 애니메이션) + 실시간 미리보기
  - **검색** — `#키워드` 복사 시 자동 검색, 결과 클릭 복사
  - **원어 사전** — 단어 클릭(히브리어 48px 헤드워드+음역), 마우스 hover 미리보기, 우클릭 새 창, 한/영 전환
  - **본문 수동 복사**(절 클릭/드래그) · 본문↔원어 스크롤 동기화 · 키보드 ←/→ 장 이동
  - 글자 크기 A±, 다크 모드(달 아이콘 채움/비움), 마지막 위치·창 크기/위치 기억
  - 업데이트 확인 + 인앱 자동 설치(다운로드→교체→재시작)
- **번들 폰트** — Pretendard 포함. **Windows는 WebView2(Win11 기본 탑재), macOS는 내장 WKWebView** 사용(별도 런타임 불필요).
- 내부: UI 비의존 `Library` 코어로 로직 단일화(데스크톱/웹 공용), 헤드리스 테스트 추가.
